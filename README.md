# 🌿 AgriScan — Drone Based Autonomous Crop Monitoring System

A deep learning system for plant disease detection across 57 disease 
classes using MobileNetV2 transfer learning.

## Features
- 🔍 Single and batch leaf image scanning
- 🌤 Live weather + crop disease risk forecasting
- 🗂 Multi-field management with scan history
- 📊 Officer dashboard for disease trend monitoring
- 📄 PDF export for scan reports

## Model
- Architecture: MobileNetV2 (transfer learning)
- Dataset: PlantVillage + PlantDoc + Rice + Wheat (79,898 images)
- Classes: 57 disease classes across 14 crops
- Validation Accuracy: 84.6%

## Tech Stack
- Backend: Python, Flask, TensorFlow 2.19
- Frontend: HTML, CSS, JavaScript, Chart.js
- Deployment: Google Colab + Cloudflare Tunnel
